About Project

This project uses Artificial Intelligence and Deep Learning techniques to detect diseases in sugarcane leaves. It helps farmers identify issues early and reduce crop loss.

Key Features

  • Upload sugarcane leaf image
  • CNN-based disease detection
  • Fast & accurate prediction
  • Suggested solutions for treatment

Technologies Used

Frontend: HTML, CSS, JavaScript
Backend Framework: Django (Python)
Deep Learning: Convolutional Neural Network (CNN)
Libraries: TensorFlow, Keras, NumPy, OpenCV, Matplotlib
API Communication: JSON, Fetch API
Database: SQL (MySQL)

How It Works

  1. User uploads sugarcane leaf image
  2. Image preprocessing
  3. CNN model analyzes the image
  4. Disease is detected
  5. Solution is displayed
